# Simple Graphic Editor

Simple graphic editor in Java made as a school project. 

Allows users to place different objects on the canvas or to have them placed randomly.

It is possible to delete objects, change their color, move them or have object groups.

## Functions
Below are functions of given buttons in app:

* **Square, Rectangle, Triangle, Oval** - Place a selected shape

* **Random square, rectangle, triangle, oval** - Select to place selected object to a random place on screen, object will have randomized size attribute

* **Change Color** - use to change color of a selected object

* **Select** - select an object

* **Save** - save image as `.png` into project folder
